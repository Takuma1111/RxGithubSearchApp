# RxGithubSearchApp
RxSwiftを用いてGithubのリポジトリ検索アプリを作りました。
検索したリポジトリを押下するとそのリポジトリのWebページに飛びます。

使用したライブラリ
 ・RxSwift
 ・RxCocoa
 ・FontAwesome_swift
 
 実行過程
 ターミナルにてこのプロジェクトファイルに移動して<code>$ bundle i</code>をします。
 そしてRxGithubSearch.xcworkspaceのファイルを開きます。
